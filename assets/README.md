# `assets/` - Models and Scenes

This folder contains MuJoCo assets used by the project.

## Subfolders

- `assets/scenes/`
  - project-specific Panda + table scenes used by run scripts
- `assets/mujoco_menagerie/`
  - vendored third-party asset bundle (kept mostly untouched)
- `assets/franka/`
  - placeholder directory for extra Franka-specific assets

For active experiments in this repository, use `assets/scenes/panda_table_scene.xml`.
